Pro odhad maker z fotky potřebuješ Anthropic API klíč.

Klíč se ukládá lokálně v prohlížeči (localStorage) a posílá se pouze na api.anthropic.com. Vytvořit klíč: console.anthropic.com → API Keys. Pozor: v prohlížeči je klíč viditelný v JS sources. Pro osobní zařízení OK, pro produkční použití nasaď proxy (Cloudflare Worker / vlastní backend).
